# blog-source
Source code of my blog



# 上传操作

```git
git add .
git commit -m "ADD new post"
git push origin main
```

# 下载操作

新电脑要下载博客源码并进行操作有如下几步：

1. 安装nodejs

2. 拉取源码

   ```git
   git clone git@github.com:Hoshea99/blog-source.git
   cd blog-source
   ```

3. 安装依赖

   ```git
   npm install
   ```

4. 安装hexo命令行工具

   ```git
   npm install -g hexo-cli
   ```
